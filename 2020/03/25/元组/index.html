简述

python中元组与列表类似，只是元组是不可修改的类型，如果内嵌列表，则内嵌部分可以修改，元组是一种数据类型，英文为tuple

创建元组

元组创建很简单，只需要在括号中添加元素，并使用逗号隔开即可

元组运算符

Python表达式 运行结果 描述
len((1, 2, 3)) 3 计算元素个数
(1, 2, 3) + (4, 5, 6) (1, 2, 3, 4, 5, 6) 连接
(‘Hi!’,) * 4 (‘Hi!’, ‘Hi!’, ‘Hi!’, ‘Hi!’) 复制
3 in (1, 2, 3) True 元素是否存在
